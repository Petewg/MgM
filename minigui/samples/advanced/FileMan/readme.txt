What is FileMan ?

The FileMan is a FREEWARE file manager for Win9x, Windows 2000 and Windows XP.
It has a graphical interface with two file panels and combinations of keys like
Norton Commander. The FileMan allows copying, move, delete, make files and
directories, edit and view files, change and set attributes for files.

Requires: Typical Windows Installation.

Made by Harbour and MiniGUI.

Enjoy ! :-)
