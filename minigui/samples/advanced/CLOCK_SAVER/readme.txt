DESCRIPTION:

This freeware screen saver shows an analog clock.
Have a lot of fun! 

INSTALLATION: 
~~~~~~~~~~~~~
Run the file "ClockSaver.exe". The ClockSaver.scr will be
copied into your Windows folder, and then it will be set up.
