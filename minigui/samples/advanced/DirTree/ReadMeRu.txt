DirTree

Программа позволяет построить дерево файлов для выбранного каталога (диска).
Можно указать шаблон включаемых файлов и рассматривать архивы как каталоги. В последнем случае в построение включаются всё содержание архива без учёта маски.

Средствами самой программы обрабатываются архивы ZIP. Архивы других типов раскрываются 7-Zip. При этом программа выбирает один из возможных методов вызова 7-Zip:
- обращение к установочной версии
- вариант размещения 7z.exe и 7z.dll в каталоге самой программы
- наличие в каталоге программы консольной версии 7za.exe (поддерживает меньшее количество типов архивов)

После построения структуры можно открыть текущий элемент:
- для не заахивированных папок и файлов выполняется команда "Открыть"
- запакованные файлы извлекаются в системный каталог временных файлов и также выполняется попытка открытия. После закрытия программы просмотра извлечённые файлы.

!!! При просмотре элемента дерева главное окно DirTree блокировано.


Недостатки.

- для просмотра вызывается программа, ассоциированная с типом файла, поэтому открываются не все файлы.
- файлы с аттрибутом "Скрытый" для просмотра не открываются (функция File(), вызываемая перед открытием для проверки их не видит)
