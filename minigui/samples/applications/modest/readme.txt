DESCRIPTION

In contrast to utility DBU, the program allows:

- to change designation and type of field without the necessity
  for the preliminary unloading of data;
- to determine the formula of the conversion of field's contents
  at the restructuring;
- to preserve an arbitrary descriptions database and fields.

Simultaneously program makes it possible to work only with one structure,
but it is possible to launch the several copies of application.
