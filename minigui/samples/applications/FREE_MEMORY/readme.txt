  FreeMemory is 32-bit Win9x/NT/2000/XP/Vista/7
freeware program that reclaims RAM from the operating system
and applications. This decreases load time and improves
application performance.

Features:
~~~~~~~~
 Frees up RAM manually or automatically
 Easily free specified amount of RAM
 Automatically or manually tunes system file cache
 Uses system tray
