	-------------------------------------------------------------------------
	Intro
	-------------------------------------------------------------------------
        Simple program that opens the "SendTo" folder of the current user.
        The location of this folder can be obtained from the function
        SHGetSpecialFolderLocation(). Then program tries to convert the Item
        Identifier to a string using SHGetPathFromIDList() and opens the folder
        using ShellExecute().


	-------------------------------------------------------------------------
	Requirements
	-------------------------------------------------------------------------
	Typical Windows Installation.


	-------------------------------------------------------------------------
	Installation
	-------------------------------------------------------------------------
	Put the program folder anywhere you like.
	Delete the program folder and it contents to uninstall.


	-------------------------------------------------------------------------
	Usage
	-------------------------------------------------------------------------
	Doubleclick.
