What is Password Generator?
===========================

This free application lets you generate hard-to-crack passwords.
The generated passwords can easily be customized to fit your needs.

Features:

- Set password length;
- Don't use the upper letter O in the generated passwords;
- Force to use or don't use the digits (easier to memorize but easier to crack).

Note: the recommended password is at least 10 characters long.
