Desktop Notes v.1.1

Description:
============
This small program allows you to attach virtual
post-it notes to your windows desktop easily.
You can choose an individual visual appearance
for each note.

Not overloaded with huge number of features
it has nice and intuitive interface.

For quick access program puts an icon in the system tray.
To add a new note just click the program icon in the tray.
Existing note can be edited from its context menu.

Features:
=========
- Tray icon for easy operation
- Selectable colors, fonts for each note from its edit window 
- You can set individual notes "always on top" from context menu of note.

Requirements:
=============
Windows 98, ME, Windows NT 4.0, Windows 2000, Windows XP
