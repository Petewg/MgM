What is DBFview ?

The DBFview is a FREEWARE user-friendly Viewer and Editor for DBF files.
You can view and edit in OEM(DOS) or Windows character sets, get information
about fields, filter, find, replace, add, delete/recall, pack, zap records.

Requires: Typical Windows Installation.

Installing: Just unzip the files to a directory on your hard disk.

Use: Run "dbfview.exe" or use command line: "dbfview [basename]" 

Made by Harbour and MiniGUI.

Please report for all problems or suggestions to the author.

Enjoy ! :-)
