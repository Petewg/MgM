EasyInstall v1.1
----------------

EasyInstall is a 32-bit program for Windows95/98/NT designed
for the creation of setup procedures for applications.

The installer allows uncompress setup.zip, create Menu groups
and desktop icon and reboot the computer after the installation
if this is necessary. You can show your logo in BMP, JPG or GIF.

The simple config file setup.ini used for the control of setup
process.

EasyInstall is freeware.

This program can be distributed on CD-ROM, Floppy Disks, WWW etc.
Please freely distribute.
