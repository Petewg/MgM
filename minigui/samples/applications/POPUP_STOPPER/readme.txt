Platforms: Windows 9x/Me/NT4/2000/XP
Status: Freeware


Description
~~~~~~~~~~~
Program place icon at icon tray. At first the program remember all windows, that have
"Microsoft Internet Explorer" or "Internet Explorer" in header and then close
an any new window with the same headers.
When you click icon, it will disactivate/activate program.
If program runs with /beep parameter, it will play a sound on every window closing.


Installation
~~~~~~~~~~~~
Just place program at any folder that you want.
