DESCRIPTION:
~~~~~~~~~~~~
Uninstall Editor displays the entries in the Add/Remove Programs
list of Windows Registry, checks if they are valid, and uninstalls
the selected program or removes the entry from the list.

INSTALLATION: 
~~~~~~~~~~~~~
To install this program unzip to any directory/folder, create
a shortcut by right clicking the file UninstallEditor.exe with
your mouse then select Create shortcut. Cut and paste the shortcut
to you desktop and rename to anything you like.
The installation is complete.
