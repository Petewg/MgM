DESCRIPTION:
~~~~~~~~~~~~
This freeware program minimizes the Outlook Express to system
tray by one mouse click on the tray icon.

You can starting the Outlook Express also if this program is not
started already.

If the program is started with the command line parameter
/hide or -hide then it will be run the Outlook Express in latency.
